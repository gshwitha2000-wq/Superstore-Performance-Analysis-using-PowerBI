# 🛍️ Superstore Retail Performance Analysis Dashboard

## 📌 Overview

The **Superstore Retail Performance Analysis Dashboard** provides a detailed view of sales, profit, and operational performance across different regions, categories, and customer segments. It enables businesses to identify trends, optimize logistics, and improve profitability using data-driven insights.

---

## 🚀 Key Features

### 📊 Business Overview

* **Total Sales:** 2M
* **Total Quantity Sold:** 22K
* **Total Profit:** 29.70
* **Average Delivery Days:** 3.93
* **Total Cities Covered:** 452

## 📈 Key Insights

* **Consumer segment** contributes the highest sales.
* **California and New York** dominate in both sales and profit.
* **Online payments** are the most preferred transaction method.
* **Phones and Chairs** are among the top-selling sub-categories.
* Delivery performance is efficient with an average of under 4 days.

## ▶️ How to Use

1. Open the dashboard in Power BI / Tableau.
2. Use region filters (Central, East, South, West) to explore data.
3. Hover over charts to view detailed metrics.
4. Analyze trends to improve sales and operational strategies.
